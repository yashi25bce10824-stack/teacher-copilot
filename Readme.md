🤖 AI Teacher Copilot

An AI-powered web application that acts as a virtual teacher, providing instant, clear, and simplified answers to student queries.

⸻

🚀 Problem Statement

Students often struggle to understand concepts due to:
	•	Complex explanations in textbooks
	•	Lack of instant doubt-solving
	•	No personalized learning support

⸻

💡 Solution

AI Teacher Copilot provides:
	•	Real-time answers to questions
	•	Simplified explanations using AI
	•	An interactive and easy-to-use interface

It acts like a 24/7 personal tutor.

⸻

🧠 How It Works
	1.	User enters a question in the frontend
	2.	The request is sent to the backend (Flask server)
	3.	Backend processes the request and sends it to the AI model
	4.	AI generates a response
	5.	The response is sent back and displayed to the user

⸻

🛠️ Tech Stack
	•	Frontend: HTML, JavaScript
	•	Backend: Python (Flask)
	•	AI Integration: OpenAI API
	•	Other Tools: Flask-CORS

⸻

⚙️ Installation & Setup

1. Clone the repository
git clone <your-repo-link>
cd teacher
2. Install dependencies
pip install flask flask-cors openai
3. Add your API key
client = OpenAI(api_key="")
4. Run the backend
python app.py
5. Open frontend

Open index.html in your browser.
